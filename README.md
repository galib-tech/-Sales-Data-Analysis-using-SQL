# 📊 Sales Data Analysis using SQL

🚀 **A complete end-to-end SQL project focused on extracting business insights from a real-world sales dataset.**

---

## 🧠 Project Overview

This project demonstrates how raw sales data can be transformed into meaningful insights using SQL.
The dataset consists of multiple relational tables including customers, products, sales transactions, calendar, territory, and returns.

The goal of this project is to simulate real-world business analysis by answering key analytical questions using SQL.

---

## 🗂️ Dataset & Structure

The project is built on a structured database containing:

* 🧑‍💼 Customer data (demographics, income, occupation)
* 📦 Product data (category, price, subcategory)
* 🛒 Sales transactions (orders, quantity, dates)
* 🌍 Territory/region data
* 🔁 Returns data
* 📅 Calendar data (date, month, year)

---

## 🔧 Data Preparation

Before analysis, the dataset was cleaned and standardized:

* Converted string dates into proper SQL `DATE` format
* Modified column data types for consistency
* Ensured relational integrity across tables

---

## 📈 Key Business Questions Solved

### 🔹 Basic Analysis

* Retrieve all sales records
* Identify unique products
* Filter customers by income, occupation, and demographics
* Count total products and orders

### 🔹 Intermediate Analysis

* Find highest-priced products
* Identify oldest customers
* Analyze category-wise product distribution
* Calculate total orders and sales

### 🔹 Advanced Analysis 🔥

* 💰 Total revenue by region
* 📦 Product-wise total sales quantity
* 🏆 Top 5 revenue-generating products
* 🔁 Return analysis and return percentage
* 👥 Identify repeat customers
* 📊 Monthly sales trends
* 🥇 Ranking products using `RANK()` and `ROW_NUMBER()`
* 📌 Top products by category

---

## 🛠️ SQL Skills Demonstrated

* ✅ Data Cleaning & Transformation
* ✅ SELECT, WHERE, DISTINCT
* ✅ JOIN (Multiple table joins)
* ✅ GROUP BY & Aggregations
* ✅ ORDER BY & LIMIT
* ✅ Date Functions
* ✅ CTE (Common Table Expressions)

## 📊 Key Insights (Example)

* High-revenue products significantly drive overall sales performance
* Certain regions consistently outperform others in total sales
* A small group of repeat customers contributes heavily to revenue
* Return rates highlight potential issues in specific product categories
